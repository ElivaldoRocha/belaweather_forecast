Este projeto está sendo desenvolvido como um portfólio para demonstrar minhas habilidades como desenvolvedor full stack Python. O objetivo é criar aplicações que acessam dados matriciais, processam informações e desenvolvem ferramentas úteis para a análise de dados meteorológicos e divulgação de informações. Este projeto não tem fins lucrativos.

As imagens utilizadas são do Multi-Source Weather (MSWX), um produto meteorológico operacional de alta resolução (a cada 3 horas com 0,1°), corrigido por viés, com cobertura global desde 1979 até 7 meses a partir de agora. Outros produtos meteorológicos, como ERA5, HydroGFD, PGF e WFDE5, não estão disponíveis em quase tempo real, não possuem previsões gratuitas e têm uma resolução espacial grosseira (≥0,25°). O MSWX combina as melhores fontes de dados para cada escala temporal e elimina vieses sistemáticos para fornecer uma solução eficaz e prontamente disponível para uso em aplicações de modelagem operacional. Além disso, o MSWX é compatível com o produto de Precipitação de Conjunto Ponderado Multi-Fonte (MSWEP) da GloH2O, que combina dados de medidores, satélites e reanálises para obter as melhores estimativas de precipitação.

Estou utilizando previsões de médio prazo, que são inicializadas diariamente às 0 horas UTC (equivalente às 21 horas de Brasília) e têm um horizonte de previsão de 10 dias. O site será atualizado todos os sábados com a previsão e boletins semanais.
